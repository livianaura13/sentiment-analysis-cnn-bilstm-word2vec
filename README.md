# Sentiment Analysis using CNN-BiLSTM with Attention

This project implements a hybrid deep learning model for sentiment classification using CNN, BiLSTM, and attention mechanisms.

## Key Results
- Achieved F1-score: 93.77%
- Improved performance by +2.72% compared to baseline
- Evaluated on:
  - IMDB Dataset (50,000 reviews)
  - Twitter Airline Dataset (14,000+ tweets)

## Key Features
- Hybrid deep learning architecture (CNN + BiLSTM)
- Self-attention & global attention mechanisms
- Performance comparison on short vs long text

## Tech Stack
- Python
- TensorFlow / PyTorch
- NLP (Tokenization, Lemmatization)
- Deep Learning (LSTM, CNN, Attention)

## Business Insight
This model can be used for:
- Customer sentiment analysis
- Product review analysis
- Social media monitoring
